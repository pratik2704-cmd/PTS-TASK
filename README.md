## ☁️ Deployment

### Deploy on Vercel

```bash
npm install -g vercel
vercel
```

### Deploy on Netlify

```bash
npm run build
```

Upload the generated `dist/` folder.

### Deploy on GitHub Pages

Install:

```bash
npm install gh-pages --save-dev
```

Add inside `package.json`:

```json
"scripts": {
  "deploy": "gh-pages -d dist"
}
```

Deploy:

```bash
npm run build
npm run deploy
```
